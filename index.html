About Me

The Developer Who Automates Everything

I'm Nicholas Knight, a full-stack developer and entrepreneur who builds solutions that work while you sleep. What started as curiosity about making computers do tedious tasks has evolved into a passion for creating sophisticated automation systems that solve real-world problems.

I code for the joy of it. There's something magical about writing a script at midnight and waking up to see it processed hundreds of files, uploaded content, and generated reports while I slept. My recent projects reflect this passion - from video processing pipelines that automatically capture game highlights, compress them, and upload to Discord, to YouTube automation that converts videos to 9:16 format and generates AI-powered titles using OpenAI.

My approach? The best code is code that runs itself. Whether it's Python scripts monitoring game events, API integrations connecting Discord/YouTube/OpenAI seamlessly, or web applications that adapt to user behavior in real-time - I approach every project asking "How can this be completely automated?"

From relaunching Knight Group LLC with modern digital infrastructure to building custom e-commerce platforms and sophisticated gaming automation suites, I bring both technical expertise and business acumen to every challenge.
